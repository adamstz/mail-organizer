# organize-mail

Lightweight scaffold for an email processing app.

Structure:

- `backend/` - Python FastAPI backend, Gmail Pub/Sub subscriber, LLM integration.
- `frontend/` - React frontend placeholder.
- `llm/` - Local LLM deployment helpers, model wrappers and composition.
- `docs/` - Architecture notes and runbooks.

This repo is an abstract scaffold to iterate on. See each folder README for next steps.
